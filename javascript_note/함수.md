# 함수

같은 동작을 하는 코드들, 중복되는 동일한 동작의 코드들을 하나로 묶어서 사용할 때 함수를 사용한다.

<br />

같은 동작을 하는 코드들, 중복되는 동일한 동작의 코드들을 하나로 묶어서 사용할 때 함수를 사용한다.

```jsx
function test(parameter) {}

test(argument);
```

### 매개변수 (parameter)

함수를 선언할때 input자리에 들어가는 변수를 매개변수 혹은 parameter라고 부름

### 인자 (argument)

함수를 호출할 때 넣어 주는 값을 인자 혹은 argument라고 부름
